## Personal PowerShell scripts for media manipulation

Contains hard-coded paths/opinionated options, requires manual review before use

Software required to be present in PATH:

- Deno
- FFmpeg
- FFprobe
- Gallery-dl
- MediaInfo
- Yt-dlp
- ImageMagick
- LosslessCut
- MPC-HC
- GPAC

Optional:

- QAAC
- LibreOffice Calc
